# ✂️ Rock Paper Scissors

A simple and fun browser-based Rock Paper Scissors game where you play against the computer. Built using pure HTML, CSS, and JavaScript — no frameworks, no libraries.

## 🎮 How to Play

1. Click on **Rock**, **Paper**, or **Scissors**
2. The computer randomly picks its choice
3. The winner is decided instantly
4. Scores are tracked for both you and the computer

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
